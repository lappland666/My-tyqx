# 椭圆曲线相关计算（C++实现）

## 项目简介
本项目实现了有限域上的椭圆曲线相关运算，包括点加法、倍点运算以及点阶计算等。

## 核心功能

### 基础算法
- 快速幂（ksm）
- 扩展欧几里得算法（求逆元）
- 模运算

### 椭圆曲线运算
1. 判断是否为椭圆曲线
2. 判断点是否在曲线上
3. 椭圆曲线点加法（P + Q）
4. 倍点运算（mP）
5. 点的阶计算
6. 曲线的阶计算
7. 枚举曲线上的所有点

## 文件结构

- `1.cpp`：主程序
- `mymath.cpp / mymath.h`：数学运算实现
- `README.md`：项目说明
